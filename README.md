# DiskLens 磁盘洞察

> 面向 Windows 的磁盘空间分析、全系统快速搜索与安全清理工具。本地运行、便携发布,用接近专业桌面工具的速度和体验,告诉你磁盘空间花在了哪里、又能怎样安全地腾出来。

## 功能概览

磁盘洞察围绕三件事展开 —— **看清占用、找到文件、安全清理**,对应界面左侧的三个核心模块:

- 📊 **磁盘分析** —— 扫描整盘,用目录树、内容表格和空间占比图,把"空间去了哪"讲清楚。
- 🔍 **文件搜索** —— 全盘索引后按文件名 / 路径秒级检索,支持分页加载海量结果。
- 🧹 **垃圾清理** —— 按安全等级分组给出可清理项,支持移入回收站、可撤销。

## 核心功能

### 一、磁盘空间分析

扫描某个磁盘或目录后,从多个视角定位空间占用大户:

- **目录树**:按目录层级展开,每个节点显示自身及子项的累计占用,一眼看出哪个文件夹最占地方。
- **目录内容表**:进入任一目录后,列出其中的文件与子目录、大小、类型,支持排序,快速锁定大文件。
- **空间占比图(Treemap)**:把目录结构画成嵌套矩形,面积即占用大小 —— 最大的一眼可见,不必逐层翻找。
- **大文件榜**:整盘 / 整目录里最大的若干文件单独成表,清理目标直指最大头。
- **类型统计**:按扩展名聚合(如 `.mp4`、`.iso`、`.zip`),看清是哪一类文件在吃硬盘。
- **疑似重复**:识别可能重复的文件,辅助发现冗余备份。

**两种扫描模式,自动选择**:

- **NTFS 极速扫描**:管理员权限下直接读取 NTFS 主文件表(MFT),整盘秒级完成。
- **兼容扫描**:无管理员权限或非 NTFS 卷时,自动回退到多线程目录枚举,保证可用。
- 模式切换对用户透明 —— 极速模式不可用时会自动降级,无需手动干预。

### 二、全系统文件搜索

- 为固定磁盘建立本地索引,之后按文件名或路径关键字检索,响应在毫秒到秒级,不必每次重新遍历整盘。
- 结果分页加载,即便命中海量文件也不卡顿。
- 纯本地索引,不联网、不上传任何文件名。

### 三、安全清理

不是"一键全删",而是把可清理项**按风险分组**,让你清楚每一项的性质再决定:

- **安全项**:缓存、临时文件等可放心清理的内容。
- **谨慎项**:删除前值得看一眼的,清理器会提示确认。
- **高级项**:需要你了解后果后再处理的内容。
- 支持**普通清理**与**深度清理**两档力度。
- 默认移入**回收站**、可撤销,避免误删难以挽回。

### 四、界面与体验

- **三套主题**:浅色专业(默认)、暗色大师、蓝色清爽,菜单一键切换并记住选择。
- **自绘图标**:界面图标全部由 QPainter 手绘,不使用任何第三方产品图标资源。
- **高分屏 / 多显示器友好**:支持 PerMonitorV2、分数缩放下取整渲染、多显示器图标清晰,4K 与 150% 缩放下不发糊。

## 系统要求

- **操作系统**:Windows 10(1809 及以上)/ Windows 11,64 位(源自 Qt 6.8 运行时要求,详见下文「兼容性说明」)。
- **权限**:NTFS 极速扫描需管理员权限(程序已声明 `requireAdministrator`);无权限时自动降级为兼容扫描。
- **运行时**:动态链接 MSVC 运行时,目标机需装有 VC++ Redistributable;便携包已随附所需 DLL。

## 快速开始

1. 获取程序:从 [Releases](https://github.com/sunnyfancore/DiskLens/releases) 下载发布包(若已发布),或按下方步骤自行构建。
2. 运行 `磁盘洞察.exe`(会请求管理员权限,用于极速扫描)。
3. 在左侧选择模块:**磁盘分析** / **文件搜索** / **垃圾清理**。

## 构建(开发者)

需要 Visual Studio 2022(或更新)与 Qt 6 MSVC 64 位套件:

```powershell
cmake -S . -B build-qt -G "NMake Makefiles" -DCMAKE_BUILD_TYPE=Release -DBUILD_QT_UI=ON -DCMAKE_PREFIX_PATH=E:\Qt\6.8.3\msvc2022_64
cmake --build build-qt --target DiskLens
```

产物为 `build-qt\磁盘洞察.exe`;`scripts\package-release.ps1` 可进一步生成便携包与单文件包。

## 兼容性说明

- **超长路径(>260 字符)**:扫描深层目录(如 `node_modules`、`.git`)时启用 `\\?\` 前缀,不再静默漏扫;占用统计同步修正。
- **应用清单**:嵌入完整清单,声明 Windows 10/11 `supportedOS`、`PerMonitorV2` DPI 感知、`longPathAware` 长路径支持、公共控件 v6,并保留管理员提权。
- **为何仅支持 Win10/Win11**:源于 Qt 6.8 运行时,而非本项目限制;若需支持 Windows 7/8,需降级到 Qt 5.15 LTS 重新编译。
