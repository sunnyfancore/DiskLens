# DiskLens 磁盘洞察

> 面向 Windows 的磁盘空间分析、全系统快速搜索、内容去重、安全清理与硬盘健康监测工具。本地运行、便携发布,用接近专业桌面工具的速度和体验,告诉你磁盘空间花在了哪里、有哪些重复文件可回收、硬盘还健康吗,又能怎样安全地腾出空间。

## 功能概览

磁盘洞察围绕**五大能力**展开 —— **看清占用、找到文件、回收重复、安全清理、保障硬盘**:

- 📊 **磁盘分析** —— 扫描整盘,用目录树、内容表、空间占比图和类型统计,把"空间去了哪"讲清楚。
- 🔍 **文件搜索** —— 全盘索引后按文件名 / 路径秒级检索,支持按时间、大小、类型筛选。
- ♻️ **重复去重** —— 用 SHA-256 逐字节比对找出**内容真正相同**的文件(含跨名副本),勾选后一键回收。
- 🧹 **垃圾清理** —— 按安全等级分组给出可清理项,默认移入回收站、可撤销。
- 🩺 **磁盘健康** —— 读取每块物理盘的 SMART / NVMe 健康日志,看温度、通电时长和寿命评估。

下面逐项说明怎么用。

## 核心功能

### 一、磁盘空间分析

扫描某个磁盘或目录后,从多个视角定位空间占用大户:

- **目录树**:按目录层级展开,每个节点显示自身及子项的累计占用,一眼看出哪个文件夹最占地方。
- **目录内容表**:进入任一目录后,列出其中的文件与子目录、大小、类型与修改时间,支持点列头排序。
- **空间占比图(Treemap)**:把目录结构画成嵌套矩形,面积即占用大小 —— 最大的一眼可见,不必逐层翻找。
- **大文件榜**:整盘 / 整目录里最大的若干文件单独成表,清理目标直指最大头。
- **类型统计**:按扩展名聚合(如 `.mp4`、`.iso`、`.zip`),看清是哪一类文件在吃硬盘。
- **长期未动**:按修改时间列出最久没碰过的文件,找出长期占地的"僵尸"数据。

**两种扫描模式,自动选择**:

- **NTFS 极速扫描**:管理员权限下直接读取 NTFS 主文件表(MFT),整盘秒级完成。
- **兼容扫描**:无管理员权限或非 NTFS 卷时,自动回退到多线程目录枚举,保证可用。
- 模式切换对用户透明 —— 极速模式不可用时会自动降级,无需手动干预。

### 二、全系统文件搜索

- 为固定磁盘建立本地索引,之后按文件名或路径关键字检索,响应在毫秒到秒级,不必每次重新遍历整盘。
- **高级筛选**(搜索栏下方筛选条):可按**时间范围**(今天 / 近 7 天 / 近 30 天 / 近一年 / 自定义)、**大小**(<10 MB / 10–100 MB / 100 MB–1 GB / >1 GB)、**类型**(仅文件 / 仅目录)叠加过滤,精确缩小结果。
- 结果分页加载,即便命中海量文件也不卡顿;点列头可按名称、大小、类型、修改时间排序。
- 纯本地索引,不联网、不上传任何文件名。

### 三、重复文件内容去重

不是按"同名"猜,而是**按真实内容**找出重复:

- **快速视图**:即时按"同名 + 同大小"列出候选,无需读取内容。
- **内容深度校验**:点按钮后用 **SHA-256** 三段式比对(先按大小分组 → 硬链接自动归并 → 前 16KB 头哈希分桶 → 全文哈希确认),逐字节确认真正相同的文件,**跨名副本也能发现**。后台执行、可随时取消。
- 树形展示:每个去重组列出所有副本及其路径、大小、修改时间;可勾选"每组保留首项,其余待删"。
- **删除方式**:默认移入**回收站**(可撤销);勾选"永久删除"则直接删除(不可逆)。受保护路径(系统关键目录)会被自动拦截。

### 四、安全清理

不是"一键全删",而是把可清理项**按风险分组**,让你清楚每一项的性质再决定:

- **安全项**:缓存、临时文件等可放心清理的内容。
- **谨慎项**:删除前值得看一眼的,清理器会提示确认。
- **高级项**:需要你了解后果后再处理的内容。
- 支持**普通清理**与**深度清理**两档力度。
- 默认移入**回收站**、可撤销,避免误删难以挽回。

### 五、磁盘健康监测

读取每块物理硬盘的健康信息,在数据出问题前提醒你:

- 每块物理盘一行,显示**型号、接口、容量、健康度、温度、通电时长、状态**。
- **NVMe 固态盘**:读取 SMART/Health 日志,显示综合磨损度(寿命已用百分比)、可用备用块、温度、通电时长与次数。
- **SATA / 机械盘**:通过 SMART 直读关键属性(重映射扇区、当前待映射扇区、离线无法校正扇区、温度、通电时长 / 次数)。
- 状态分四级:**良好 / 注意 / 警告 / 不可读取**,异常盘的健康度和状态文字会变色高亮。
- 后台读取、可取消;读不到的盘会显示"不可读取"而不会影响其他盘。

### 六、导出与报表

分析结果可导出留档或交给他人:

- 支持 **CSV**(纯表格,可直接用 Excel / WPS 打开)与 **HTML**(带样式的报表,浏览器直接查看)两种格式。
- 在任意分析页(目录内容、大文件、类型统计、疑似重复、长期未动、快速搜索、垃圾清理)按 `Ctrl+E` 或右键导出当前结果。
- 导出字段完整,包含名称、大小、类型、路径与修改时间。

### 七、界面与体验

- **四套导航模块**:磁盘分析、文件搜索、重复去重、垃圾清理、磁盘健康一键切换(`Ctrl+1` ~ `Ctrl+8`)。
- **三套主题**:浅色专业(默认)、暗色大师、蓝色清爽,菜单一键切换并记住选择。
- **自绘图标**:界面图标全部由 QPainter 手绘,不依赖任何第三方产品图标 / 字体资源。
- **高分屏 / 多显示器友好**:支持 PerMonitorV2、分数缩放下取整渲染,4K 与 150% 缩放下不发糊。

## 注意事项

- **管理员权限**:程序已声明 `requireAdministrator`,每次启动会请求提权 —— 这是 NTFS 极速扫描与物理盘 SMART 直读所必需的;不提权时扫描会自动降级、健康页会读不到物理盘信息。
- **去重删除**:默认走回收站、可撤销;勾选"永久删除"后**不可恢复**,请确认后再用。
- **SMART 读取范围**:NVMe 与直连 SATA/ATA 盘可读;USB 桥接盘、部分 RAID/虚拟盘可能不支持 SMART 直读,会显示"不可读取"而非报错。
- **超长路径(>260 字符)**:深层目录(如 `node_modules`、`.git`)扫描时启用 `\\?\` 前缀,不再静默漏扫。

## 系统要求

- **操作系统**:Windows 10(1809 及以上)/ Windows 11,64 位(源自 Qt 6.8 运行时要求)。
- **权限**:启动即请求管理员权限(用于极速扫描与 SMART 直读);无权限时自动降级。
- **运行时**:动态链接 MSVC 运行时,目标机需装有 VC++ Redistributable;便携包已随附所需 DLL。

## 快速开始

1. 获取程序:从 [Releases](https://github.com/sunnyfancore/DiskLens/releases) 下载发布包(若已发布),或按下方步骤自行构建。
2. 运行 `磁盘洞察.exe`(会请求管理员权限)。
3. 在左侧选择模块:**磁盘分析** / **文件搜索** / **垃圾清理** / **磁盘健康**;「疑似重复」入口在磁盘分析模块的标签页内。
4. 常用快捷键:`Ctrl+E` 导出、`Ctrl+R` 重新扫描、`Ctrl+1`~`Ctrl+8` 切换模块、`F1` 查看快捷键。

## 构建(开发者)

需要 Visual Studio 2022(或更新)与 Qt 6 MSVC 64 位套件:

```powershell
cmake -S . -B build-qt -G "NMake Makefiles" -DCMAKE_BUILD_TYPE=Release -DBUILD_QT_UI=ON -DCMAKE_PREFIX_PATH=E:\Qt\6.8.3\msvc2022_64
cmake --build build-qt --target DiskLens
```

产物为 `build-qt\磁盘洞察.exe`;`scripts\package-release.ps1` 可进一步生成便携包与单文件包。

## 兼容性说明

- **应用清单**:嵌入完整清单,声明 Windows 10/11 `supportedOS`、`PerMonitorV2` DPI 感知、`longPathAware` 长路径支持、公共控件 v6,并保留管理员提权。
- **为何仅支持 Win10/Win11**:源于 Qt 6.8 运行时,而非本项目限制;若需支持 Windows 7/8,需降级到 Qt 5.15 LTS 重新编译。
